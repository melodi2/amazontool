# amazontool
Keyword Pro is a React.js-based single page application where users can upload their Report table (CSV file), select criteria for keyword optimization, and receive analysed results.

## Features

-   Upload CSV Report file
-   Select Keyword Optimization criteria
-   Receive/Display results


## Technology Stack

-   React.js, Redux
-   Node.js, Express
-   PostgresSQL
-   JavaScript

With the uploader, the user can choose the CSV file with the newest data they want to upload. 
In the next step, they choose the criteria to filter out the Winning or Losing Keywords. Finally, the user gets the analyzed Results, which can be used to optimize the advertising campaign.
